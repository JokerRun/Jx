

## 1.ods_prod

### backup

```MYSQL
mysqldump  --add-drop-table -hlocalhost -uroot -p123456@ ods_prod | gzip > ods_prod.sql.gz
# ls -lh #查看输出文件大小
#  scp ./ods_prod.sql.gz 101.227.69.51:/root/
```



### recovery

```mysql
# cd /root/
gunzip < ods_prod.sql.gz | mysql -hlocalhost -uroot -p123456@ ods_prod
```
